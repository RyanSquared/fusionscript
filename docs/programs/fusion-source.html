Script fusion-source

Run FusionScript code with the Lua VM.

Usage:

    fusion-source [OPTIONS] [FILE]
fusion-source [FILE]

  --metadata <package> | Print metadata for package
  -m <package>         | Run <package>.main module
  -h                   | Print help information
